# Изображения для галереи работ

Замените эти изображения на реальные фотографии ваших работ.

## Рекомендации:
1. Размер превью: 600x400 пикселей
2. Размер больших фото: 1200x800 пикселей  
3. Формат: JPG с качеством 80%
4. Имена файлов:
   - bmw-wheels.jpg          - Диски BMW
   - bmw-wheels-large.jpg    - Увеличенное фото
   - metal-gate.jpg          - Кованые ворота
   - etc...

## Где взять изображения:
1. Сделать фото своих реальных работ
2. Использовать стоковые фото (Unsplash, Pexels)
3. Нанять фотографа для съемки

Пример поиска на Unsplash:
- Порошковая покраска: https://unsplash.com/s/photos/powder-coating
- Автомобильные диски: https://unsplash.com/s/photos/car-wheels
- Металлические конструкции: https://unsplash.com/s/photos/metal-structures
